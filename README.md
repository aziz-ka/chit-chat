# [ChitChat Client](https://chitchat.now.sh)


## Stack

- React
- Firebase
- Socket.io
- Bootstrap 4
- Now


#### Quick Start

```shell
$ git clone git@github.com:aziz-ka/chitchat-client.git
$ cd chitchat-client
$ yarn
$ yarn start
```


#### Yarn/NPM Commands

|Script|Description|
|---|---|
|`yarn start`|Start the app on `localhost:3000`|
|`yarn build`|Build the app to `./build`|
